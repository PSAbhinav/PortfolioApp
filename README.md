# Portfolio App

## 📱 Overview
Portfolio App is an Android application built using **Kotlin and Jetpack Compose** that showcases a personal portfolio in a clean and modern interface. The app presents user information, projects, and other details in a structured layout suitable for demonstrating Android UI design skills.

## ✨ Features
- Modern UI built with Jetpack Compose
- Multiple composable screens
- Clean and responsive layout
- Organized project information display

## 🛠️ Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio
- Material Design Components

## 📂 Project Structure
- `MainActivity.kt` – Entry point of the application
- `AboutScreen.kt` – Displays portfolio information
- `ui.theme` – Theme and styling components

## 🚀 How to Run
1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle
4. Run the app on an emulator or physical device

## 📸 Output
The app displays a professional portfolio layout with sections for personal information and project details.

## 📚 Learning Outcome
This project helped in understanding **Jetpack Compose layouts, composable functions, and UI structuring in Android development.**
